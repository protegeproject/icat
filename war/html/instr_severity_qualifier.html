Important Note: A value for the severity property must be specified if and only if this category represents a specific sub-category of a parent category, which is subdivided by a severity scale.

  Example: A value for the severity property must be specified for
    "Type 1 diabetes mellitus with mild nonproliferative ophthalmic complications"
    "Type 1 diabetes mellitus with moderate nonproliferative ophthalmic complications"
    "Type 1 diabetes mellitus with severe nonproliferative ophthalmic complications"
  but NOT for their parent category
    "Type 1 diabetes mellitus with ophthalmic complications"
  or any other category that does not have a severity modifier (such as "Type 1 diabetes mellitus", "Lipodystrophies", etc.).
